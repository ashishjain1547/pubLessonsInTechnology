# TASK
Spin off a transcript into a blog post.

# WORD LIMIT
3000 words

# STRUCTURE
- Organize blog post in sections and headings.
- Use exact quotations from the transcript that you find really, really interesting and important.
- Have a "Conclusion" section at the end written in bullets.

# TONE AND LANGUAGE
Explanatory tone with layman's language

# VISUAL ELEMENTS
- You can use tabular representation of data to compare and contrast in a better way.
- And you are also free to draw plots/graphs of your choice for a good data story.

# DOS AND DON'TS
- Do not mention that you are reading from any transcript or video.
- Write it as if you are the one making these points/predictions/statements (which are coming out from the transcript).

# INPUT
- Attached transcript (.txt) file

# OUTPUT
- Return a clean, formatted HTML for the blog post

# OUTPUT DESCRIPTION
- Wrap the contents of the post in a <div> with class="customOuterWrapper"
- Write CSS rules starting with "div.customOuterWrapper ..."
- Do not write any generic / global CSS rule.
- Do not use special characters for list bullets. Use keyboard symbols / characters only.
